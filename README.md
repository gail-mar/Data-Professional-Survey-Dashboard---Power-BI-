Power BI dashboard analyzing global data professional survey data to identify salary disparities, in-demand skills, and satisfaction trends highlighting key gaps between career expectations and reality.

## 🧠 The Real Question Behind This Project
What actually drives success and satisfaction in data careers—and where are people getting it wrong? This dashboard breaks down salaries, skills, and sentiment to uncover the gap between expectation vs reality in the data field.

### 🔥 Key Insights (That Actually Matter)
- 💰 Data Scientists dominate salaries—but the gap isn’t just skill, it’s positioning
  → Roles closer to business impact get paid more
- 🐍 Python isn’t just popular—it’s a baseline requirement
  → If you don’t know Python, you’re already behind
- 🚧 Breaking into data is the biggest bottleneck—not the work itself
  → High barrier to entry is filtering talent early
- 🙂 Work-life balance is decent, but salary satisfaction lags
  → People aren’t burnt out—they feel underpaid
- 📉 Lower-tier roles (Analyst, DB Dev) are where dissatisfaction concentrates
  → These roles risk high turnover

### 🧩 What This Dashboard Covers
- Salary distribution by job role
- Programming language preferences
- Global respondent distribution
- Perceived difficulty entering data careers
- Happiness scores (work-life vs salary)

### 🛠️ Tools & Skills Demonstrated
- Power BI – dashboard design, DAX measures, data storytelling
- Data interpretation – turning survey data into actionable insights
- UX thinking – structuring visuals for clarity and decision-making

### 🎯 So What? (Why This Actually Matters)

This isn’t just a dashboard—it highlights real risks and opportunities:
Companies:
Underpaying analysts while overvaluing titles → leads to churn
Aspiring Data Professionals:
Focusing only on tools instead of impact → limits earning potential
Teams:
Ignoring satisfaction gaps → long-term retention problems
